## DEV
1. renombrar el `.env.template` a `.env`.
2. Configurar variables de entorno.
3. Ejecutar el comando 
```
docker compose up -d
```
4. Ir al localhost:3001 y ver que funcione bien.
5. Comprobar que la DB también esté funcionando.
